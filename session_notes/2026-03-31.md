### 1. Implementation: Sample Splitting Method ✓

**File created**: `package/R/sample_splitting_minimax_wasserstein.R` (425 lines)

**Core functions:**
- `sample_splitting_minimax_wasserstein()` - Main inference function
- `split_data()` - Stratified 50-50 split helper
- `identify_worst_case_d1()` - Find optimal gamma* on D1 (identification)
- `infer_on_d2()` - Estimate concordance on D2 (inference)
- `bootstrap_ci_sample_splitting()` - Bootstrap CI on D2 only

**Algorithm:**
1. Split data: D1 (identification), D2 (inference) — independent samples
2. D1: Find worst-case region → optimal dual variable γ*
3. D2: Estimate concordance using γ* from D1 (no selection bias!)
4. Bootstrap D2 only → valid CI

**Theoretical guarantee** (to be proven in Theorem 1):
```
Under regularity conditions (smoothness, bounded moments, overlap):
  - Consistency: φ̂_split →^p φ*(λ_w)
  - Asymptotic normality: √n₂(φ̂_split - φ*) →^d N(0, σ²)
  - Bootstrap validity: Bootstrap on D2 is consistent
  - Coverage guarantee: P(φ* ∈ CI) → 1-α
```

**Key innovation**: D1 ⊥ D2 → no post-selection bias → standard M-estimation theory applies

## Key Decisions

### 1. Sample Splitting Design
- **50-50 split default** (can adjust via `split_ratio` parameter)
- **Stratified by treatment** to maintain balance
- **Fixed gamma* from D1** used for all bootstrap samples on D2
- **Cross-fitting optional** within each split (reduces bias in tau estimation)

### 2. Bootstrap Strategy
- **Resample D2 only** (not D1) → no selection bias
- **500 bootstrap samples** (default)
- **Percentile CI** (standard, easy to interpret)
- **Parallel option** available for speed

### 3. Implementation Choices
- **Multiple tau methods** supported (linear, kernel, RF, GAM)
- **Multiple cost functions** (euclidean, manhattan)
- **Reproducible** via seed parameter
- **Verbose option** for monitoring progress

## Theoretical Foundation

### Regularity Conditions (to be formalized in Theorem 1)
1. **Smoothness**: τ_S(x), τ_Y(x) are Lipschitz continuous
2. **Bounded moments**: E[τ_S(X)^k], E[τ_Y(X)^k] < ∞ for k=2,4
3. **Overlap**: P(A=1|X=x) ∈ [ε, 1-ε] for some ε > 0
4. **Covariate support**: X has bounded support or exponential tails

### Proof Strategy (Theorem 1)
- **Independence**: D1 ⊥ D2 → selection on D1 doesn't affect D2
- **M-estimation**: φ̂ on D2 is solution to smooth estimating equation
- **Standard theory**: van der Vaart & Wellner (1996) weak convergence
- **Bootstrap**: Classical bootstrap consistency (Efron & Tibshirani 1993)
