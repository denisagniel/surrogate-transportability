#### 1. Efficient Influence Function Derivation (3.5 hours)

**Two-stage structure:**

**Stage 1: Treatment Effect Influence Functions**
- **RCTs:** Weighted means with IF: ψ_ΔS(O; Q, P₀) = w(Q, O) [A(S - μ₁)/π - (1-A)(S - μ₀)/(1-π)]
- **Observational studies:** AIPW with cross-fitting (doubly robust):
  - ψ_AIPW(O; Q, P₀) = w(Q, O) [A(S - μ̂₁(X))/ê(X) - (1-A)(S - μ̂₀(X))/(1-ê(X)) + μ̂₁(X) - μ̂₀(X) - ΔS(Q)]
  - Cross-fitting removes nuisance parameter bias (no stringent rate conditions)
  - Doubly robust: consistent if either e(X) or μₐ(X) is correct

**Stage 2: Functional Delta Method for Correlation**
- Correlation functional φ is Hadamard differentiable
- Influence function for correlation: ψ_cor((θS, θY); F) = [(θS - μS)(θY - μY) - ρ σS σY] / (σS σY)
- Gradient coefficients αS(Q), αY(Q) from correlation derivative

**Combined Influence Function:**
ψ_Θ(O; P₀, λ) = E_Q~μ [αS(Q) · ψ_ΔS(O; Q, P₀) + αY(Q) · ψ_ΔY(O; Q, P₀)]

**Main result:**
√n(Θ̂ₙ - Θ(P₀)) →ᵈ N(0, σ²(λ))

where σ²(λ) = E_P₀[ψ²_Θ(O; P₀, λ)]

**Variance estimation:**
- Plug-in estimator: σ̂²(λ) = (1/n) Σᵢ ψ̂²_Θ(Oᵢ; P̂ₙ, λ)
- Confidence intervals: Θ̂ₙ ± z₁₋α/₂ · σ̂(λ)/√n

**Key properties:**
1. √n-consistency (standard parametric rate)
2. Doubly-robust for observational studies
3. Cross-fitting removes nuisance bias
4. Efficient (achieves semiparametric efficiency bound under regularity)
5. Practical variance estimation via plug-in

#### 2. Reliability Coefficient Formula (30 minutes)

**Variance decomposition:**
Var_μ[ΔS(Q)] = Var_μ[E_Q[ΔS(X)]] + Var_μ[E_Q[εS(i)]]
                 \_____________/    \____________/
                  Signal (between-X)  Noise (within-X)

**Definition:**
reliability_S = σ²_signal,S / σ²_total,S = Var(signal) / Var(total)

**Interpretation:**
- High reliability (≈ 1): Most variation is systematic (transportable signal)
- Low reliability (< 0.7): Substantial idiosyncratic noise

#### 3. Observation-Level Ceiling (Attenuation Formula) (30 minutes)

**Classical measurement error result applied to our setting:**

If εS ⊥ εY (idiosyncratic noises uncorrelated), then:
ρ_obs ≈ ρ_signal × √(reliability_S × reliability_Y)

**The ceiling:** Even if ρ_signal = 1 (perfect signal correlation):
ρ_obs ≤ √(reliability_S × reliability_Y)

**Example:** With reliability_S = reliability_Y = 0.5:
ρ_obs ≤ √(0.5 × 0.5) = 0.5 (ceiling at 50%)

**Proof sketch provided** via covariance decomposition showing noise terms vanish under independence.

#### 4. X-Level vs Observation-Level Formal Distinction (20 minutes)

**X-level geometry:**
U_X(P₀, λ) = {Q over X-distributions: TV(Q_X, P₀,X) ≤ λ}
- Varies: P(X)
- Fixed: ΔS(X), ΔY(X)
- Aggregate: ΔS(Q) = Σ_x Q(x) ΔS(x)

**Observation-level geometry:**
U_obs(P₀, λ) = {Q over individuals: TV(Q, P₀) ≤ λ}
- Varies: Individual distributions (treats each as unique)
- Allows: Unmeasured U, idiosyncratic ε
- Aggregate: ΔS(Q) = Σᵢ Q(i) [ΔS(Xᵢ) + εS,i]

**When they agree:** If no within-X heterogeneity (εᵢ = 0) and no unmeasured modifiers, both yield same correlations.
