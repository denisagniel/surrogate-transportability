# Session Notes: 2026-04-17

## Task: X-Level vs Observation-Level Geometry - Conceptual Clarification

### Summary

Deep conceptual discussion about what X-level and observation-level TV ball geometries actually measure, why observation-level typically shows lower correlations, and what the "right" estimand is for surrogate transportability. **Major insight:** The estimand is ALWAYS across-study correlation, never across-type correlation. The geometry determines which set of future studies we're considering.

### Context

Working with DGPs where treatment effect heterogeneity is defined by discrete covariates X: ΔS(X), ΔY(X). When defining local geometry in X-space, we can recover high correlation between surrogate and outcome effects. But when defining geometry in n-dimensional observation space (treating each individual as unique), correlation is lower. Question: Is this a failure of the method, or are these genuinely different quantities?

### Key Discussion Points

#### 1. Initial Hypothesis: Different Spaces, Different Correlations

**First interpretation:**
- **X-space correlation:** cor(ΔS(X), ΔY(X)) across X values (type-level)
- **Observation-level correlation:** cor(τS(i), τY(i)) across individuals
- These measure different things because individuals have variation beyond X:
  ```
  τS(i) = ΔS(X_i) + εS(i)  [signal + noise]
  τY(i) = ΔY(X_i) + εY(i)
  ```

**Issue identified:** Observation-level mixes:
1. Systematic effect modification (X and unmeasured U)
2. Idiosyncratic individual noise (ε)

If εS ⊥ εY, the observation-level correlation is attenuated by uncorrelated noise.

#### 2. User's First Key Insight: X-Level Makes Strong Assumptions

**X-space assumes:** X completely determines treatment effect heterogeneity
- No unmeasured effect modifiers (U)
- Two people with same X have same effects
- Only P(X) changes across studies, not mechanisms

**Observation-level allows:** Unmeasured heterogeneity U
- X partially determines effects, U matters too
- More general transportability question
- Both P(X) and relationship with U can change

**Trade-off identified:**
- X-level: Higher correlation, but assumes strong structure (optimistic if X is good)
- Observation-level: Lower correlation, but more robust to misspecification (conservative)

#### 3. User's Second Key Insight: The Ceiling Problem

**Critical observation:** Observation-level has a ceiling on possible correlation!

**The problem:** All individual variation (signal + noise) gets treated as if it matters for transportability, but:
- Systematic effect modification (ΔS(X,U)) matters for aggregate effects ✓
- Idiosyncratic noise (εS(i)) averages out in large samples ✗

**For aggregate effects:**
```
ΔS(Q) = E_Q[τS(i)] = E_Q[ΔS(X,U)] + E_Q[εS(i)]
                    → E_Q[ΔS(X,U)]  [with large n]
```

**But observation-level measures:**
```
cor(τS(i), τY(i)) = attenuated by noise variance
```

Even if true transportable correlation is perfect (cor(ΔS(X,U), ΔY(X,U)) = 1.0), observation-level can only recover:
```
cor_obs ≈ √[Var(signal_S)/Var(total_S)] × √[Var(signal_Y)/Var(total_Y)]
```

This is the **reliability** - the signal-to-total variance ratio. With substantial noise, hits a ceiling well below 1.0.

**Conclusion:** Observation-level confounds transportable signal with non-transportable noise.

#### 4. My Response: Within-Sample Shortcut Question

**Question raised:** How is this different from just estimating cor(ΔS(X), ΔY(X)) within the current sample?

**Initial thinking:** For discrete X, you can directly compute:
```r
Delta_S[x] = E[S | A=1, X=x] - E[S | A=0, X=x] for each x
Delta_Y[x] = E[Y | A=1, X=x] - E[Y | A=0, X=x] for each x
cor(Delta_S, Delta_Y)  # Under current P0(X)
```

**What TV ball adds:**
- Characterizes how correlation varies across different Q distributions
- Not just P0, but all plausible future X distributions
- Provides uncertainty quantification, worst-case bounds

#### 5. User's Fundamental Reframing: Always Across-Study!

**CRITICAL INSIGHT:** The estimand is NEVER "across types," it's ALWAYS "across studies."

**Example with K=2:**
```
Type 1: ΔS(1) = 0, ΔY(1) = 1
Type 2: ΔS(2) = 1, ΔY(2) = 0

"Correlation across types" = ??? (only 2 points!)

But "correlation across STUDIES" is well-defined:
  Study Q1 (90% type 1): ΔS = 0.1, ΔY = 0.9
  Study Q2 (50% type 1): ΔS = 0.5, ΔY = 0.5
  Study Q3 (10% type 1): ΔS = 0.9, ΔY = 0.1

  cor([0.1, 0.5, 0.9], [0.9, 0.5, 0.1]) = -1.0  ✓
```

**The correct estimand:**
```
Θ(λ) = cor_{Q ~ μ on U(P0,λ;d)} (ΔS(Q), ΔY(Q))
```

Where:
- μ is a measure over the set U(P0, λ; d) of plausible future studies
- Each Q defines a potential future study with its own X (or individual) distribution
- We compute aggregate effects ΔS(Q), ΔY(Q) for each study
- Then correlate these aggregates **across studies**

**What geometry determines:** Which set of future studies we consider, NOT whether we measure across-type vs across-individual.

### Revised Understanding of X-Level vs Observation-Level

Both estimate the same type of quantity (across-study correlation), but differ in which studies are included:

#### X-Level Geometry
```
U_X(P0, λ) = {Q over X values: TV(Q, P0) ≤ λ}
```

**Studies differ in:** P(X) only
**Assumes fixed:** The functions ΔS(X), ΔY(X)
**Transportability question:** "Will S predict treatment effects when populations differ in covariate distributions?"
**Appropriate when:** X captures key effect modifiers, mechanisms stay constant

#### Observation-Level Geometry
```
U_obs(P0, λ) = {Q over individuals: TV(Q, P0) ≤ λ}
```

**Studies differ in:** Everything (individuals treated as unique)
**Allows:** Different mixes of individuals, including their noise/unmeasured features
**Transportability question:** "Will S predict treatment effects under most general changes?"
**Appropriate when:** Want maximum robustness, conservative bound

### Why Observation-Level Correlation Is Lower: The Mechanics

When sampling Q from observation-level TV ball:
```
ΔS(Q) = Σ_i Q(i) · τS(i)
      = Σ_i Q(i) · [ΔS(X_i) + εS(i)]
      = [Between-X signal] + [Within-X noise]
```

**Sources of variation across different Q's:**

1. **Signal variation (Between-X):**
   - Different Q's have different Q_X(x) = Σ_{i:X=x} Q(i)
   - Creates variation in Σ Q_X(x) · ΔS(x)
   - **This variation IS correlated across ΔS and ΔY** ✓

2. **Noise variation (Within-X):**
   - Different Q's can weight individuals within same X differently
   - Creates variation in Σ_i Q(i) · εS(i)
   - **If εS ⊥ εY, this variation is UNCORRELATED** ✗
   - Inflates denominator without adding to numerator

**Attenuation formula:**
```
cor_obs(ΔS(Q), ΔY(Q)) = Cov(signal_S, signal_Y) /
                        √[(Var(signal_S) + Var(noise_S)) · (Var(signal_Y) + Var(noise_Y))]

                      < Cov(signal_S, signal_Y) / √[Var(signal_S) · Var(signal_Y)]

                      = cor_signal(ΔS(Q), ΔY(Q))
```

**The ceiling:**
```
cor_obs ≈ cor_signal × reliability_S × reliability_Y
```
where reliability = Var(signal) / Var(total)

This is exactly classical measurement error attenuation!

### Is the Lower Correlation a Bug or Feature?

**Perspective A: It's a Bug (Noise Shouldn't Matter)**

Argument:
- For large-sample transportability, individual noise averages out
- Future studies will have many observations, so E[εS(i)] → 0
- Should only care about systematic effects ΔS(X) or ΔS(X,U)
- Observation-level is misleadingly pessimistic

**Implication:** Use X-level geometry (or de-bias observation-level)

**Perspective B: It's a Feature (Real Worst-Case)**

Argument:
- Future studies could have different mixes of individuals, not just X
- Small sample sizes where noise doesn't average out
- Genuine unmeasured heterogeneity U that looks like noise
- Observation-level provides conservative bound

**Implication:** Report observation-level as conservative estimate

### Variance Decomposition Approach

Can distinguish signal from noise empirically:

```r
# For each Q sampled from observation-level TV ball:
signal_S[m] = sum(Q_X[x] * Delta_S[x])  # Between-X component
noise_S[m] = Delta_S_total[m] - signal_S[m]  # Within-X component

# Compute:
Var_signal_S = var(signal_S)
Var_noise_S = var(noise_S)
reliability_S = Var_signal_S / (Var_signal_S + Var_noise_S)

# De-bias if needed:
cor_debiased = cor_obs / √(reliability_S × reliability_Y)
```

**If reliability ≈ 1:** Noise negligible, obs-level ≈ X-level (robust conclusion)
**If reliability < 1:** Noise dominates, gap reflects attenuation (X-level more informative)

### Proposed Approach for Paper

**1. Compute both:**
   - X-level: cor(ΔS(Q), ΔY(Q)) with Q over X distributions
   - Observation-level: cor(ΔS(Q), ΔY(Q)) with Q over individuals

**2. Decompose variation:**
   ```
   Var(ΔS(Q))_obs = Var(between-X) + Var(within-X)
   reliability = Var(between-X) / Var(total)
   ```

**3. Report:**
   - X-level correlation + interpretation: "Under compositional changes (P(X) shifts)"
   - Observation-level correlation: "Under more general changes (including individual resampling)"
   - Reliability: "Fraction of variation from between-X vs within-X"

**4. Interpret:**
   - High reliability (≈1): Both similar, robust conclusion
   - Low reliability (<1): Gap from noise, X-level more informative for large-sample transportability
   - Gap = cor_X - cor_obs reflects noise attenuation + unmeasured heterogeneity

### Key Learnings

**[LEARN:estimand-clarity]** The estimand for surrogate transportability is ALWAYS across-study correlation cor(ΔS(Q), ΔY(Q)) where Q varies over plausible future studies. It's never "across types" - the types/covariates are computational tools, not the target of inference.

**[LEARN:geometry-interpretation]** X-level vs observation-level geometry determines WHICH studies are in the set U(P0, λ; d), not whether we measure individual vs aggregate quantities. Both measure across-study correlation, but over different sets of possible studies.

**[LEARN:attenuation-ceiling]** Observation-level correlation has a ceiling due to noise variation from within-X reweighting. The ceiling is cor_signal × √reliability, where reliability = Var(signal)/Var(total). This is real attenuation, not measurement error.

**[LEARN:signal-noise-decomposition]** Can empirically decompose Var(ΔS(Q)) into between-X (signal) and within-X (noise) components. Reliability coefficient tells you how much the observation-level correlation is attenuated. High reliability → robust; low reliability → X-level more informative.

**[LEARN:transportability-assumptions]** X-level geometry assumes mechanisms transport (ΔS(X), ΔY(X) functions stay constant). Observation-level allows more general changes but conflates noise with signal. For large-sample transportability where noise averages out, X-level is usually more appropriate.
