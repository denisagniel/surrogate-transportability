## Implementation Summary

### New Files Created

**1. R/tv_ball_sampling.R**
- `sample_tv_ball()`: Hit-and-run MCMC sampler for TV ball B_λ(P₀)
- Ported from `explorations/tv_ball_geometry/01_hit_and_run_sampler.R`
- Samples M future study distributions Q from {Q : TV(Q, P₀) ≤ λ}
- Default: M=500, burn_in=1000, thin=10

**2. R/gradient_correlation.R**
- `gradient_correlation_analytical()`: Analytical gradient ∇φ for correlation
- Formula: ∂ρ/∂Δ_S(Q_m) and ∂ρ/∂Δ_Y(Q_m) at each study m
- Returns M × 2 matrix of gradients (influence function of sample correlation)
- `gradient_correlation_numerical()`: Numerical validation via finite differences

**3. R/tv_ball_correlation_IF.R**
- `tv_ball_correlation_IF()`: Main inference function (RCT-only)
- Algorithm:
  1. Sample M distributions from TV ball
  2. Compute treatment effects via weighted difference-in-means
  3. Compute correlation across M studies
  4. Compute gradient at each study
  5. Compute AIPW IFs (simplified for e=0.5)
  6. Compose via chain rule: ψ_Θ(O_i) = Σ_m [∇φ(Q_m) · (ψ_S(O_i;Q_m), ψ_Y(O_i;Q_m))]
  7. Variance: σ̂² = (1/n) Σᵢ ψ_Θ(O_i)²
  8. CI: ρ̂ ± z_{α/2} √(σ̂²/n)

**4. tests/testthat/test-tv-ball-correlation-IF.R**
- Validates analytical vs numerical gradient (correlation > 0.999, MAE < 0.002)
- Tests gradient properties (sum to zero = centered IF)
- Tests TV ball sampler validity (row sums = 1, TV(Q,P₀) ≤ λ)
- Tests full inference function on RCT data
- Tests input validation and edge cases

**5. validation/validate_tv_ball_correlation_IF.R**
- Demonstration script with 3 examples:
  - Simple RCT with diagnostic plots
  - Coverage simulation (50 reps)
  - Effect of lambda on estimates and CI width

## Key Design Decisions

**1. RCT-Only First**
- Simplified implementation: no propensity scores, no outcome regressions
- Just weighted difference-in-means with e=0.5
- Easier to validate, faster to implement
- AIPW extension deferred to future work

**2. Analytical Gradient**
- Exact formula for ∂ρ/∂(Δ_S, Δ_Y) at each observation
- Validated against numerical differentiation (correlation > 0.999)
- ~100× faster than numerical gradient (no additional MCMC runs)

**3. Centered Influence Functions**
- IFs automatically centered (sum to ~0) by construction
- Key property for asymptotic normality
- Verified in tests

**4. Default Parameters**
- M = 500: Good balance (MCMC error ~0.045 SE, ~10 sec runtime)
- burn_in = 1000: Sufficient for K ≤ 100, λ ≤ 0.5
- thin = 10: Standard thinning to reduce autocorrelation

## Theoretical Properties (from proof)

**Asymptotic Normality:**
√n(ρ̂ - ρ(μ_M)) →_d N(0, σ²(μ_M))

where σ²(μ_M) = E[ψ_Θ²] is variance of composite IF

**Rate:** √n-consistent (semiparametric efficient for RCTs)

**MCMC Approximation:**
- Fixed M: Bias O_P(M^{-1/2}) independent of n
- M → ∞: Unconditional asymptotic normality for ρ(μ)

### TV Ball Geometry for Discrete X

**Binary X:**
- TV distance: TV(Q, P₀) = |q - p₀| where q = P_Q(X=1)
- TV ball: B_λ(P₀) = [max(0, p₀-λ), min(1, p₀+λ)]
- Example: P₀(X=1) = 0.5, λ = 0.3 → Ball = [0.2, 0.8]

**K-ary Discrete X:**
- TV ball is convex L₁-ball on simplex
- Shape: Diamond/cross-polytope restricted to Δ^{K-1}
- More flexible but harder to interpret

**Choice: Binary X** (simplest, sufficient for validation)

## Three-Method Comparison Implementation

### Goal
Implement and compare three approaches for TV ball correlation inference:
1. **Importance weighting** (explicit w_i = Q(X_i)/P₀(X_i))
2. **Bootstrap resampling** (resample with probabilities Q_m)
3. **AIPW** (augmented IPW with Q_m-specific nuisance estimates)

### Context
Previous findings suggested importance weighting gave near-perfect correlation while true correlation was much lower. Hypothesis: poor CATE estimation in small samples.

### Phase 2: Three-Method Implementation (COMPLETE)

**Modified:** `R/tv_ball_correlation_IF.R`

**New parameter:** `method = c("bootstrap", "importance_weighting", "aipw")`

**Implementation details:**

**1. Importance Weighting:**
- Explicit weights: w_i = Q_m(X_i) / P₀(X_i)
- Weighted group means for treatment effects
- AIPW IF with importance weights
- Fast (~same as bootstrap)

**2. Bootstrap Resampling:**
- Resample n observations with probabilities Q_m
- Compute effects on resampled data
- Weighted IF with normalized weights
- Fast (original implementation)

**3. AIPW (with Q_m-specific nuisance refitting):**
- **Key decision:** Refit nuisances for each Q_m (user request)
- Cross-fitting with Q_m weights for training
- Estimate e(X), μ₁(X), μ₀(X) under Q_m distribution
- Full three-term AIPW IF
- Slow (M × n_folds model fits = ~2500 fits for M=500, K=5)

**AIPW refitting rationale:**
- Each Q_m represents different study distribution
- Nuisances should be estimated under target distribution
- More theoretically rigorous than fitting once under P₀
- Computationally expensive but correct

**Storage:**
- Pre-allocate M sets of nuisance estimates (e_hat_all, mu_1S_hat_all, etc.)
- Use Q_m-specific estimates in IF computation
- No additional importance weights needed (already in nuisances)

## DGP 5: Small ΔY at P₀, High Correlation (PTE Failure Case)

### Goal
Add DGP demonstrating PTE failure when ΔY ≈ 0 in reference study, but correlation remains high.

### User Request
"Let's add an example where ΔY is very small in the current study, but there is high correlation across studies. PTE will really struggle if ΔY is near 0."

### Implementation

**File created:** `validation/create_dgp5_small_delta_y.R` (223 lines)

**DGP design:**
- **Antisymmetric treatment effects**: τ_S(X) = 0.5×X, τ_Y(X) = 0.3×X + 0.6×τ_S(X)
- **Symmetric reference distribution**: P₀ = [0.05, 0.25, 0.40, 0.25, 0.05] on X ∈ {-2,-1,0,1,2}
- **Key property**: Both ΔS(P₀) and ΔY(P₀) ≈ 0 by symmetry

**Parameters:**
```r
gamma_A = 0.0      # No effect at X=0
gamma_AX = 0.5     # τ_S scales with X
beta_A = 0.0       # No direct effect at X=0
beta_AX = 0.5      # Direct effect scales with X
beta_S = 0.6       # Moderate mediation
beta_SX = 0.0      # No S×X interaction
```

**Results:**
- ΔS(P₀) = 0.0015 (≈ 0)
- ΔY(P₀) = 0.0025 (≈ 0)
- **PTE(P₀) = 0.368 (UNDEFINED - division by ~0)**
- **ρ = 0.999996 (near-perfect correlation)**

**Interpretation:**
- PTE completely fails when ΔY ≈ 0 (numerator and denominator both ≈0)
- But correlation is well-defined and near-perfect
- Both effects scale linearly with X → perfect transportability
- When distribution shifts (Q ≠ P₀), both ΔS and ΔY change proportionally
