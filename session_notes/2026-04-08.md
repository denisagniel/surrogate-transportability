## Phase 1: DGP 1 - Non-Mediated Heterogeneity (COMPLETE)

### Summary

Implemented first comprehensive DGP demonstrating scenarios where traditional surrogate methods (PTE, mediation, correlation) can fail while TV ball method works correctly. DGP 1 creates non-mediated heterogeneity: treatment affects S and Y through **separate pathways** with no S→Y causality.

### Key Scientific Insight

**The Problem:**
- Traditional methods assume S mediates or predicts Y
- But in many real scenarios, S and Y respond to treatment independently
- High within-study cor(S,Y) can arise from confounding (not surrogate quality)
- PTE and mediation analysis are artifacts when no S→Y pathway exists

**Why This Matters:**
- Practitioners may approve surrogates based on PTE or mediation
- But if effects transport independently, surrogate won't generalize
- TV ball method directly tests transportability: cor(ΔS, ΔY) across studies

### Implementation

**1. DGP Function: `generate_nonmediated_heterogeneity()` (~240 lines)**
   - Location: `R/data_generators.R`
   - Supports discrete types (K types with type-level effects)
   - Supports continuous heterogeneity (smooth CATE functions)
   - Parameters control:
     - `correlation_across`: True cor(τ_S, τ_Y) across types (target: 0.05)
     - `correlation_within`: Within-study cor(S,Y) from confounding
     - `confounding_strength`: Unmeasured confounder U effect strength
   - Returns: data, ground truth, scenario metadata

**Key Design:**
```r
# S responds to treatment (no Y influence)
S <- baseline + A * tau_S[type] + confounding_strength * U

# Y responds to treatment (no S influence)
Y <- baseline + A * tau_Y[type] + confounding_strength * U

# Result: cor(S,Y) high within study, but cor(τ_S, τ_Y) low across types
```

## Phase 1: DGP Development - Learning from Failures (IN PROGRESS)

### Summary

Attempted to implement DGP 1 (Non-Mediated Heterogeneity) but discovered through theoretical calculation that the concept **cannot produce the desired failure scenario**. Explored alternative mechanisms through theoretical derivations.

### Critical Discovery: Initial DGP Concept Doesn't Work

**What we attempted:**
- DGP where S and Y respond to treatment through separate pathways (no S→Y causality)
- cor(τ_S, τ_Y) ≈ 0 across types
- High within-study cor(S,Y) from confounding U
- Expected: High PTE, but actually poor transportability

**What we learned (via theoretical calculation):**

```
THEORETICAL RESULT:
If cor(τ_S, τ_Y) = 0, then PTE ≈ 0.31 (LOW, not HIGH)

Proof:
  - In treated arm: High S doesn't predict high Y (50/50 split)
  - β_S (coef on S in Y~A+S) only captures confounding (≈0.16)
  - Indirect effect = Δ_S × β_S = 1.0 × 0.16 = 0.16
  - PTE = Indirect/Total = 0.16/0.50 = 0.31

Fundamental issue:
  For PTE to be HIGH: need β_S large
  For β_S large: S must predict Y
  For S to predict Y: need cor(τ_S, τ_Y) > 0

→ Non-mediated heterogeneity (cor≈0) CANNOT produce high PTE
```

**Simulation confirmed:** PTE = 0.311 (matches theory exactly)

### Theoretical Explorations

**1. Effect Modification V1 (S×X interaction in Y model)**
   - File: `explorations/theoretical_pte_effect_modification.R`
   - DGP: Y depends on S×X interaction, so mediation varies with X̄
   - Result: **Failed** - Δ_S constant across studies (cor undefined)
   - Issue: Only Δ_Y varies with X̄, not Δ_S

**2. Effect Modification V2 (Both effects depend on X)**
   - File: `explorations/theoretical_pte_effect_modification_v2.R`
   - DGP: Both Δ_S(X̄) and Δ_Y(X̄) depend on covariate distribution
   - Used opposite-signed interactions: γ_AX > 0, β_AX < 0
   - Result: **Partial success**
     - ✓ High PTE in current study: 1.10 (> 0.6)
     - ✗ High correlation across studies: 0.93 (want < 0.3)
   - Issue: Mediation dominates, overwhelming the direct effect variation

### Key Insights from Today's Work

**What doesn't work:**
1. ❌ **Non-mediated heterogeneity:** If effects uncorrelated across types, PTE will be low
2. ❌ **Effect modification (simple):** If only Δ_Y varies, can't compute cor(Δ_S, Δ_Y)
3. ⚠️ **Effect modification (both vary):** Can get high PTE, but mediation creates high correlation

**Why this is hard:**
- PTE is high when S strongly mediates Y (high β_S)
- If β_S is high, then Δ_S and Δ_Y tend to be correlated
- Breaking this correlation while maintaining high PTE requires careful design

**Possible paths forward:**
1. **Tune parameters** in V2 approach (weaken mediation, strengthen direct effect variation)
2. **Types with selection bias** - Current study over-samples high-mediation types
3. **Multiple pathways** - S mediates, but other mediators vary across studies
4. **Context-dependent mechanisms** - S→Y pathway exists in current study but breaks in others

### What We Learned About Research Process

**Value of theoretical calculation FIRST:**
- Saved time by proving non-mediated heterogeneity can't work (before implementing)
- User correctly identified we built wrong thing (all methods said "poor surrogate")
- Theory → Simulate → Implement is correct order

**DGP design is harder than expected:**
- Creating high PTE with low transportability requires subtle mechanisms
- Need both within-study and across-study properties to align correctly
- May need multiple iterations to find working DGP concept
