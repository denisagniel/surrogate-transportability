### Phase 2: Type (i) Validation - Nominal Coverage Under Correct Model

#### **Script 16: Probability Functional Validation**
- **File:** `sims/scripts/16_probability_functional_validation.R`
- **Functional:** φ_prob(F_λ; ε_S, ε_Y) = P(ΔY > ε_Y | ΔS > ε_S)
- **Parameters:**
  - 1000 replications per scenario
  - 4 threshold combinations
  - 2000 studies for true φ computation
  - 1000 innovations for method
- **Expected:** 95% coverage, calibrated SEs
- **Key Question:** Does method correctly estimate conditional probability?

#### **Script 17: Conditional Mean Functional Validation**
- **File:** `sims/scripts/17_conditional_mean_validation.R`
- **Functional:** φ_cond(F_λ; δ) = E[ΔY | ΔS = δ]
- **Parameters:**
  - 1000 replications per scenario
  - 4 conditioning values: δ_S ∈ {0.1, 0.3, 0.5, 0.7}
  - Kernel-weighted ground truth
  - Silverman's bandwidth selection
- **Expected:** 95% coverage, low RMSE
- **Key Question:** Does method correctly estimate conditional mean function?

---

### Phase 3: Type (ii) Validation - Minimax Bounds Over μ Class

#### **Script 19: Minimax Validation Across All Functionals**
- **File:** `sims/scripts/19_minimax_all_functionals.R`
- **Purpose:** Test minimax bounds [φ_*, φ*] for ALL four functionals
- **Parameters:**
  - 50 replications per (λ, functional) combination
  - Tests 8 Dirichlet concentrations: α ∈ {0.01, 0.1, 0.5, 1, 2, 5, 10, 100}
  - 20-point Dirichlet grid for minimax search
  - Vertices included (max 30)
- **Expected:** 100% coverage by construction (worst-case guarantee)
- **Key Question:** Do bounds contain TRUE φ for ALL μ in class M?
- **Comparison:** Extends script 13 from correlation-only to all functionals

---

### Phase 4: Type (iii) Validation - Model-Free TV Distance Guarantees

#### **Script 20: TV-Robustness Validation**
- **File:** `sims/scripts/20_tv_robustness_validation.R`
- **Purpose:** **CRITICAL TEST** - Do bounds hold for Q generated by ANY mechanism?
- **Parameters:**
  - 100 replications per (λ, functional, mechanism)
  - 5 Q-generation mechanisms:
    - mixture (baseline)
    - covariate_shift (reweighting)
    - selection (selection bias)
    - extreme (point mass on extremes)
    - adversarial (maximize φ within TV constraint)
  - 500 studies for empirical φ(Q)
- **Expected:** ≥95% coverage if class M adequate
- **Key Question:** Does TV distance alone suffice for robustness?
- **Strongest Claim:** Bounds hold for ANY Q with d_TV(Q, P₀) ≤ λ, not just mixture model

---

### Phase 5: Type (iv) Validation - Cross-Functional Predictiveness

#### **Script 21: Cross-Functional Prediction**
- **File:** `sims/scripts/21_cross_functional_prediction.R`
- **Purpose:** Can one functional predict another's empirical performance?
- **Parameters:**
  - 200 replications per (λ, mechanism)
  - 3 mechanisms: mixture, covariate_shift, selection
  - Estimates ALL four functionals on each baseline
  - Measures empirical performance on all four functionals
- **Output:** 4×4 correlation matrix: φ̂_A vs. emp_B
- **Key Questions:**
  - Does φ̂_correlation predict empirical PPV?
  - Which functional is the best overall predictor?
  - Can we use ONE functional to assess transportability?
